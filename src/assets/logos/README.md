# Company Logos

Please add the following logo files to this directory:

## Required Logo Files:

1. **eli-lilly.svg** or **eli-lilly.png**
   - Download from: https://www.lilly.com/ or search "Eli Lilly logo SVG transparent"
   - Preferred: SVG format with transparent background

2. **ibm.svg** or **ibm.png**
   - Download from: https://www.ibm.com/ or search "IBM logo SVG transparent"
   - Preferred: SVG format with transparent background

3. **quantiphi.svg** or **quantiphi.png**
   - You've already added this one!
   - Make sure the filename matches what's in the component

## Logo Guidelines:
- Use SVG format when possible (better quality, scalable)
- PNG with transparent background is acceptable
- Recommended size: 500x500px or larger
- File naming: lowercase with hyphens (e.g., eli-lilly.svg)

## Alternative Sources:
- Wikipedia Commons: https://commons.wikimedia.org/
- Company press kits/media centers
- Brandfetch: https://brandfetch.com/
